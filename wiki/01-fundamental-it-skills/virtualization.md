## Learning Goals

- Understand the relationship between hypervisor, host OS, guest OS, and VM.
- Compare the roles of VMware, VirtualBox, ESXi, and Proxmox.
- Build an isolated virtual machine environment for security practice.

## Practice

- [ ] Install a virtualization platform.
- [ ] Create a Linux virtual machine.
- [ ] Record network modes: NAT, Bridge, and Host-only.

## Safety Notes

Run malware samples, vulnerable machines, and unknown tools only in isolated lab networks to avoid affecting real hosts or home networks.
