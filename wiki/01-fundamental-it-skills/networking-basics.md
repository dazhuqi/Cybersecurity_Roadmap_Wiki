# Basic Computer Networking

| Field | Value |
| --- | --- |
| Status | Not Started |
| Roadmap Area | Fundamental IT Skills |

## Learning Goals

- Understand the basic relationship between hosts, network cards, IP addresses, gateways, and DNS.
- Explain how a local machine reaches a website.
- Build foundations for later study of OSI, subnetting, protocols, and ports.

## Practice

- [ ] Use `ipconfig` or `ip addr` to inspect local network configuration.
- [ ] Use `ping` to test connectivity.
- [ ] Use `nslookup` or `dig` to query DNS resolution.
