# Cybersecurity Roadmap Wiki

This is the home page for the learning wiki. Use it as the roadmap index, review entry point, and long-term knowledge base.

The wiki follows the public [roadmap.sh Cyber Security Roadmap](https://roadmap.sh/cyber-security), but it is not meant to be a blind checklist. Each page should become a learning note with plain explanations, practice ideas, review questions, and references.

## How to Use This Wiki

1. Start with the current module.
2. Read the note in plain language.
3. Try one command, diagram, lab, or review question.
4. Add confusing terms to the glossary.
5. Record practice in the learning log or lab writeups.
6. Review weekly instead of opening endless new resources.

## Completion Standard for Notes

- Explain the core concept in my own words.
- List key points or common mistakes.
- Include at least one practice record, command example, diagram, or case study.
- Add review questions for later recall.
- Link official or stable references.
- Keep the Chinese version in sync when the folder supports it.
