## Learning Goals

- Practice common Windows, Linux, and macOS commands.
- Use the command line to inspect files, processes, ports, network configuration, and logs.
- Build the habit of recording command input, output, and interpretation.

## Command Notes

```bash
# Linux examples
pwd
ls -la
ps aux
ss -tulpen
```

```powershell
# Windows examples
Get-Process
Get-Service
Get-NetIPAddress
Get-EventLog -LogName System -Newest 10
```
