# Weekly Review

This page is for weekly reflection. The goal is to turn scattered study into a better next week.

## How to Use This Page

At the end of each week:

1. Check the learning log.
2. Check the progress tracker.
3. Pick the most important completed note or lab.
4. Identify one blocker.
5. Choose a small next-week focus.

## Review Questions

- Which concept can I now explain without looking it up?
- Which concept is still unclear?
- Which lab should be turned into a writeup?
- Which notes need diagrams, commands, or reference links?
- Did I practice enough, or mostly read/watch?
- Did I open too many resources?
- What should I stop doing next week?
