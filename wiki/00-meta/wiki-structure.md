# Wiki Structure

This wiki is organized as main roadmap modules plus practice modules.

## Structure Principles

- `00-meta`: Learning plans, progress, glossary, and reviews.
- `01` to `06`: Main areas aligned with the roadmap.sh Cyber Security roadmap.
- `07-practice-labs`: Labs, CTFs, and hands-on records.
- `08-certifications`: Certification path and preparation notes.
- `09-resources`: Courses, books, tools, and communities.
- `templates`: Reusable note templates.
- `assets`: Images, screenshots, and diagrams.

## New Page Rules

Add new pages to an existing module whenever possible. If a topic crosses multiple modules, choose the primary context and link to it from related pages.
