# Learning Log

This page is for short study-session records. It should be quick to update. If logging takes too long, I will stop using it.

## When to Write a Log

Write a log when I:

- finish a study session
- complete a lab room
- read a chapter or important article
- finally understand a confusing concept
- get stuck and need to preserve the blocker

## Log Format

```text
Date:
Topic:
Study Time:
Roadmap Module:
What I Practiced:
What I Understood Today:
What Is Still Unclear:
Mistake or Surprise:
Next Step:
Related Links:
```

## Quality Rule

A good log does not need to be long. It needs to answer:

- what did I do?
- what did I learn?
- what should I do next?

If the entry only says "watched videos," it is not useful enough.
