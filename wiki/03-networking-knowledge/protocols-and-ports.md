## Common Ports

| Port | Protocol | Notes |
| --- | --- | --- |
| 22 | SSH | Remote login |
| 53 | DNS | Domain name resolution |
| 80 | HTTP | Plaintext web traffic |
| 443 | HTTPS | Encrypted web traffic |
| 3389 | RDP | Windows Remote Desktop |

## Learning Goals

- Learn common ports and protocol use cases.
- Distinguish FTP/SFTP, HTTP/HTTPS, SSL/TLS, LDAPS, and IPsec.
- Understand the relationship between open ports and attack surface.
