## Recording Principles

- Record only authorized environments.
- Sanitize command output.
- Each writeup should describe objective, scope, steps, evidence, and lessons learned.
- Keep platform links here so I do not waste time searching for the same entry points again.

## Before Starting a Lab

- Confirm the platform rules and scope.
- Use a lab VM or controlled environment when possible.
- Do not attack systems outside the lab target.
- Take notes while working, not only after solving it.
- Save useful commands, mistakes, and final lessons in a writeup.
