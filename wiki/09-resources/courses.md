# Courses

Courses are useful when I need structure. The risk is watching too much and practicing too little. For each course, I should write down what I actually did after watching or reading.

## How I Pick a Course

Before starting a course, I should check:

- Does it match the module I am studying right now?
- Does it include practice, labs, or review questions?
- Can I finish a small part this week?
- Will I write notes in this wiki after each section?

If a course is famous but does not match my current topic, I should save it for later instead of opening ten tabs.

## Rule for Myself

If I finish a course section but cannot explain what I practiced, I did not really finish it. At minimum, I should record one command, one diagram, one lab result, or one review question.
