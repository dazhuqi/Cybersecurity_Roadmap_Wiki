# 08 Certifications

This directory tracks certification paths, preparation notes, and review plans.

Certifications are not the same as skill, but they can give structure. I want to use this section as a planning map, not as a shopping list. Before choosing an exam, I should ask what role it supports and whether I have enough hands-on practice to make the certificate meaningful.

## Selection Guidance

Choose certifications based on the current goal: entry-level IT, networking foundations, blue team, defense, penetration testing, governance, compliance, or senior security management.

## My Rough Order

This is not a fixed rule, but a reasonable starting order for me:

1. Build IT basics with A+ style knowledge, even if I do not sit the exam.
2. Learn networking through Network+ or CCNA-style study.
3. Learn security foundations through Security+ or GSEC-style study.
4. Choose a direction: blue team, penetration testing, cloud, audit, or management.
5. Only plan expensive or advanced exams after enough hands-on practice.
