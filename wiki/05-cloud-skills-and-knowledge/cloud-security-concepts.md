## Learning Goals

- Understand the cloud shared responsibility model.
- Understand security differences between cloud and on-premises environments.
- Understand the role of identity, networking, logging, keys, and storage configuration in cloud security.

## Review Questions

- What is the cloud provider responsible for, and what is the customer responsible for?
- Why can incorrect cloud permissions have high impact?
