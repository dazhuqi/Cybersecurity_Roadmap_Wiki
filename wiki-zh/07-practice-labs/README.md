## 记录原则

- 只记录授权环境。
- 清理命令输出里的敏感信息。
- 每篇 writeup 应该描述目标、范围、步骤、证据和学到的东西。
- 把平台链接集中放在这里，之后不用每次重新搜索入口。

## 开始实验前

- 确认平台规则和测试范围。
- 尽量使用实验 VM 或受控环境。
- 不攻击实验目标以外的系统。
- 边做边记笔记，不要只在最后补。
- 保存有用命令、踩坑点和最终经验。
