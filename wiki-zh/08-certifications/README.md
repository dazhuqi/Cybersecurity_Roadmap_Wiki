# 08 认证

这个目录用来记录认证路线、备考笔记和复习计划。

认证不等于能力，但它可以提供结构。我想把这一部分当成规划地图，而不是“想全买一遍”的清单。选择考试前，我应该先问：它支持什么岗位？我有没有足够实操，让这张证书有意义？

## 选择建议

按照当前目标选择认证：入门 IT、网络基础、蓝队、防御、渗透测试、治理、合规，或者高级安全管理。

## 我的粗略顺序

这不是硬规则，但对我来说比较合理：

1. 先建立 A+ 风格的 IT 基础，就算不考试也要补知识。
2. 用 Network+ 或 CCNA 风格学习打网络基础。
3. 用 Security+ 或 GSEC 风格学习安全基础。
4. 之后再选择方向：蓝队、渗透测试、云、审计、管理。
5. 昂贵或高级考试要等有足够实操后再计划。
