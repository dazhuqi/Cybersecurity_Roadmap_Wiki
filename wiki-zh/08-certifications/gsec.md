## 笔记

GSEC 是 GIAC Security Essentials。它是宽泛安全基础认证，但通常比纯词汇型概览更偏技术和实操。

在我的路线图里，GSEC 可以看成“安全基础”的进阶版本。它可能有价值，但在认真计划前要确认成本和培训资源是否现实。

## 为什么它和我的路线图有关

GSEC 风格学习把操作系统、网络、防御、密码学、访问控制、事件处理、安全运营连接起来。它能帮我不再把这些主题看成孤立小岛。

如果我想要扎实的蓝队或通用安全基础，它很相关。

## 我应该学到什么

- 网络安全概念。
- Defense-in-depth 和基础安全架构。
- Windows 和 Linux 安全基础。
- 认证、访问控制、密码安全。
- 密码学和 PKI 基础。
- 事件处理和响应思路。
- 安全监控和运营。

## 我会怎么准备

1. 先完成 OS 和网络笔记。
2. 建一个带例子的安全术语表。
3. 练习读取 Windows 和 Linux 日志。
4. 做小型防御实验：防火墙规则、事件日志、漏洞扫描、加固。
5. 决定前对比 GSEC 和 Security+。

## 练习想法

- 加固一台 Windows 或 Linux VM，并记录改动。
- 查看认证日志，区分正常和可疑事件。
- 把常见攻击映射到防御控制。
- 写一个小型 incident handling 流程。
