# 每周复盘

这个页面用于每周反思。目标是把零散学习变成更好的下周计划。

## 怎么用

每周结束时：

1. 检查 learning log。
2. 检查 progress tracker。
3. 选择本周最重要的一篇笔记或一个 lab。
4. 找出一个 blocker。
5. 选择一个很小的下周重点。

## 复盘问题

- 哪个概念我现在能不查资料解释？
- 哪个概念仍然不清楚？
- 哪个 lab 应该变成 writeup？
- 哪些笔记需要 diagrams、commands 或 reference links？
- 我练得够不够，还是主要在读/看？
- 我是不是打开了太多资源？
- 下周我应该停止做什么？

## 什么算进展

进展不只是完成页面，也可以是：

- 把一个主题解释得更简单
- 修正一个错误假设
- 把混乱的 lab 变成可读 writeup
- 删除一个分散注意力的资源
- 连续三次注意到同一个弱点，然后安排练习

如果一周感觉很慢，先找证据，不要马上判定失败。有时候最有用的结果只是找到真正的 blocker。

## 警告信号

- 我打开了很多 tab，但没有写笔记。
- 我看了视频，但没有做 commands 或 examples。
- 我每天都在换学习主题。
- 我避开了一个困惑点，而不是写下具体困惑。
- 我给下周安排了一个不符合真实时间的大计划。
