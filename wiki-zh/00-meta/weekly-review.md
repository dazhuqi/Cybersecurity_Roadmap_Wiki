# 每周复盘

这个页面用于每周反思。目标是把零散学习变成更好的下周计划。

## 怎么用

每周结束时：

1. 检查 learning log。
2. 检查 progress tracker。
3. 选择本周最重要的一篇笔记或一个 lab。
4. 找出一个 blocker。
5. 选择一个很小的下周重点。

## 复盘问题

- 哪个概念我现在能不查资料解释？
- 哪个概念仍然不清楚？
- 哪个 lab 应该变成 writeup？
- 哪些笔记需要 diagrams、commands 或 reference links？
- 我练得够不够，还是主要在读/看？
- 我是不是打开了太多资源？
- 下周我应该停止做什么？
