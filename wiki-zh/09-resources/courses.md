# 课程

课程在我需要结构时很有用。风险是看太多、练太少。每学一个课程，我都应该写下看完后自己实际做了什么。

## 课程列表

| 课程 | 提供方 | 主题 | 难度 | 状态 | 备注 |
| --- | --- | --- | --- | --- | --- |
| Cyber Security Roadmap | roadmap.sh | 总体路线图 | 入门 | 参考 | 用它看结构，不要当成必须立刻刷完的清单。 |
| Pre Security / Beginner Paths | TryHackMe | IT、网络、Web、安全基础 | 入门 | 未开始 | 当我需要引导式实验，而不是只读文字时很好用。 |
| Hack The Box Academy | Hack The Box | 安全模块和实操技能 | 入门到高级 | 未开始 | 想在做机器前先学解释时使用模块。 |
| Web Security Academy | PortSwigger | Web 应用安全 | 入门到高级 | 未开始 | 学请求/响应思维和 Web 漏洞很适合。 |
| Networking Basics | Cisco Networking Academy | 网络 | 入门 | 未开始 | 深入 CCNA 风格学习前可以先看。 |
| Introduction to Linux | Linux Foundation | Linux 基础 | 入门 | 未开始 | 可以作为 Linux 模块的补充。 |
| Microsoft Learn Security | Microsoft Learn | Microsoft 安全和云安全 | 入门到中级 | 参考 | 学 Windows、Defender、Entra、Azure Security 时优先看官方。 |
| AWS Skill Builder Security | AWS Skill Builder | 云安全 | 入门到中级 | 以后 | 留到云模块。 |
| OpenSecurityTraining2 | OST2 | 底层安全和逆向 | 中级到高级 | 以后 | 很好，但不是新手起点。 |

## 给自己的规则

如果我学完一个课程小节，但说不出自己练了什么，那就不算真的学完。至少要记录一个命令、一个图、一个实验结果，或者一个复习问题。
