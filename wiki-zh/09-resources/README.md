# 09 资源

这个目录收集课程、书籍、工具、网站和参考材料。

它不应该变成一个巨大的书签垃圾堆。如果我把资源加进来，就应该知道它为什么有用、适合什么水平、什么时候应该打开它。

## 页面索引

- [books.md](books.md)
- [courses.md](courses.md)
- [tools.md](tools.md)
- [websites.md](websites.md)
- [cheatsheets.md](cheatsheets.md)

## 收集原则

- 优先记录真正用过的资源。
- 写清楚适合的水平和用途。
- 避免纯粹堆链接，定期删除失效或低价值资源。
- 优先使用官方文档、primary sources 和仍在维护的参考资料。
- 标清资源属于 reference、course、practice 还是 deep reading，避免用错材料。

## 复查习惯

资源会过期。隔一段时间我应该检查：

- 链接是否还能打开？
- 资源是否还在维护？
- 我是否真的用过？
- 它对当前阶段会不会太难？
- 我有没有写下从它那里学到了什么？
